Face Recognition Application


Gender Classification: This application is powered by data and a machine learning model. When a user uploads an image, the backend automatically processes it by converting it to grayscale, detecting and cropping the face, and transforming it into the required format. The processed image is then passed to the model for gender prediction. All these functionalities are integrated within a Flask web application.
